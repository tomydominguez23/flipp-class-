# FLIP CLASS — Portal Web (tipo Skool)

Demo funcional de un portal de estudio estilo “Skool” para el curso **FLIP CLASS** (compra/venta/escala de autos en Chile).

## Qué incluye

- **Landing pública**: descripción del curso, módulos y CTA.
- **Planes**: Básico, Híbrido y Pro Automotora (según el brief).
- **Registro/Login (demo)**: sin backend (usa `localStorage`).
- **Portal `/app`**: dashboard, curso por módulos/lecciones, progreso, comunidad tipo foro y materiales descargables.

## Cómo correrlo

Requisitos: Node.js + npm.

```bash
npm install
npm run dev
```

La app vive en `flip-class/` pero puedes ejecutar desde la raíz con los scripts.
